MindScan is a web-based deep learning system that detects and classifies brain tumors from MRI scans using a lightweight hierarchical architecture. It applies a sequence of preprocessing steps to enhance image quality and consistency, then uses a two-stage MobileNetV2 pipeline: a binary model to identify tumor presence and a multi-class model to classify glioma, meningioma, or pituitary tumors. Trained on preprocessed MRI data, the system achieves high accuracy in both stages and delivers real-time predictions through an intuitive web interface, making it suitable for reliable, efficient, and accessible tumor analysis.
